# Implementacion Tiendanube - Lumen Hogar

## Objetivo
Montar tienda de luminaria + mobiliario con estructura comercial lista para vender.

## Paso a paso

1. Crear categorias:
   - Iluminacion: Colgantes, Techo, De Pie, Mesa, Pared
   - Mobiliario: Living, Comedor, Home Office, Entrada, Cocina
2. Importar `plantilla_tiendanube_lumen.csv`.
3. Configurar home por bloques:
   - Hero + CTA a catalogo
   - Categorias destacadas
   - Productos destacados (8)
   - Beneficios (envio, cuotas, soporte)
4. Configurar checkout:
   - Medios de pago
   - Medios de envio
   - Politicas y FAQ
5. Activar tracking:
   - Pixel Meta
   - GA4
6. QA final:
   - Prueba de compra completa
   - Mobile test
   - Velocidad basica

## Entrega

- Tienda publicada
- Home comercial activa
- Catalogo cargado y visible
- Checklist de lanzamiento completo
