# Kit Demo Lumen Hogar (Luminaria + Mobiliario)

Este kit sirve para mostrar un ejemplo real de implementacion para Tiendanube y Empretienda.

## Archivos incluidos

- `plantilla_tiendanube_lumen.csv`
- `plantilla_empretienda_lumen.csv`
- `colecciones_y_menu.md`
- `ofertas_y_automatizaciones.md`
- `calendario_lanzamiento_14_dias.md`

## Como usar este demo para vender tu servicio

1. Mostra `caso-lumen-hogar.html` al potencial cliente.
2. Explica que ya tenes resuelto catalogo, estructura y plan de salida.
3. Aclara que estas plantillas se adaptan en 24-48h al rubro de su negocio.
4. Cierra con propuesta por fases (setup, carga y lanzamiento).
