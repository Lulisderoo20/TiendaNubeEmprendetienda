# Implementacion Empretienda - Lumen Hogar

## Objetivo
Lanzar version Empretienda optimizada para conversion mobile y salida rapida.

## Paso a paso

1. Crear estructura de categorias:
   - Luminaria > Colgantes, Techo, De Pie, Mesa, Pared
   - Mobiliario > Living, Comedor, Home Office, Entrada, Cocina
2. Importar `plantilla_empretienda_lumen.csv`.
3. Configurar home:
   - Banner principal
   - Productos destacados
   - Categoria de ofertas
   - CTA a WhatsApp
4. Configurar pagos/envios:
   - Alias/links de pago o pasarela
   - Regla de envio gratis segun ticket
5. Configurar contenidos de confianza:
   - FAQ
   - Politica de cambios
   - Politica de envios
6. QA final:
   - Test de navegacion en celular
   - Test de checkout
   - Revisar stock y precios

## Entrega

- Tienda en vivo
- Catalogo completo cargado
- Home orientada a conversion
- Checklist final de control
